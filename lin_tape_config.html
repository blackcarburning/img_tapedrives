📦 lin_tape Driver Installation

Install and manage the IBM lin_tape kernel module on RHEL.

Check Current State

Check if lin_tape is loaded
lsmod | grep lin_tape
Check lin_tape module info
modinfo lin_tape
Check if lin_tape RPM is installed
rpm -qa | grep -i lin_tape

Download & Install

ℹ Download Location IBM Fix Central: IBM_Fix_Central_Site.html
Navigate: Product Group: Storage Systems → Tape Systems → Tape Device Drivers → Linux/Red Hat
Install lin_tape RPM
rpm -ivh lin_tape-<version>.x86_64.rpm
Build from source RPM (alternative)
rpmbuild --rebuild lin_tape-<version>.src.rpm

Blacklist Generic Drivers

⚠ Important The generic st (tape) and sg (SCSI generic) drivers conflict with lin_tape. They must be blacklisted.
File: /etc/modprobe.d/blacklist-tape-generic.conf
# Blacklist generic SCSI tape/generic drivers
blacklist st
blacklist sg
Create blacklist file (heredoc)
cat > /etc/modprobe.d/blacklist-tape-generic.conf <<'EOF'
# Blacklist generic SCSI tape/generic drivers
blacklist st
blacklist sg
EOF
Unload generic drivers
rmmod st 2>/dev/null; rmmod sg 2>/dev/null

Load & Verify

Load lin_tape module
modprobe lin_tape
Verify devices appeared
ls -la /dev/IBMtape* /dev/IBMchanger*
🔧 lin_taped Daemon Management

Install, start, stop, and monitor the lin_taped user-space daemon.

ℹ Important Notes
  • lin_taped must be stopped before unloading lin_tape (rmmod lin_tape)
  • lin_taped handles device node creation/deletion in /dev/IBMtape* and /dev/IBMchanger*
  • lin_taped provides control path failover and data path failover features
  • If tape devices are added/removed, reload the lin_tape module and restart lin_taped

Install

Check if lin_taped RPM is installed
rpm -qa | grep -i lin_taped
Install lin_taped RPM
rpm -ivh lin_taped-<version>.x86_64.rpm

Service Control

Start lin_taped (vendor command)
lin_taped start
Start lin_taped (systemd)
systemctl start lin_taped
Stop lin_taped (vendor command)
lin_taped stop
Stop lin_taped (systemd)
systemctl stop lin_taped
Restart lin_taped
lin_taped restart
Check status (vendor)
lin_taped status
Check status (systemd)
systemctl status lin_taped
Enable lin_taped at boot
systemctl enable lin_taped
Check if lin_taped process is running
pgrep -a lin_taped
🔄 Boot Persistence

Ensure lin_tape loads automatically after every reboot.

modules-load.d

Ensure lin_tape loads at boot
echo "lin_tape" > /etc/modules-load.d/lin_tape.conf

Module Options

File: /etc/modprobe.d/lin_tape.conf
options lin_tape alternate_pathing=1
Create modprobe options file
echo "options lin_tape alternate_pathing=1" > /etc/modprobe.d/lin_tape.conf

systemd Service (Vendor or Custom)

Check for vendor-provided systemd service
systemctl list-unit-files | grep lin_tape
Enable vendor service if it exists
systemctl enable lin_tape && systemctl start lin_tape
File: /etc/systemd/system/lin_tape_load.service (custom fallback)
[Unit]
Description=Load IBM lin_tape kernel module
After=systemd-modules-load.service
Before=lin_taped.service

[Service]
Type=oneshot
RemainAfterExit=yes
ExecStart=/sbin/modprobe lin_tape

[Install]
WantedBy=multi-user.target
Enable custom service
systemctl daemon-reload && systemctl enable lin_tape_load.service

initramfs

Rebuild initramfs including lin_tape
dracut --add-drivers "lin_tape" --force
Verify boot persistence
systemctl is-enabled lin_tape_load 2>/dev/null || systemctl is-enabled lin_tape 2>/dev/null
🖥️ Device Discovery

Enumerate tape drives, changers, and FC HBA ports after lin_tape is loaded.

List Devices

List tape devices
ls -la /dev/IBMtape*
List changer devices
ls -la /dev/IBMchanger*
List all SCSI devices
lsscsi -g

udev / Device Properties

Get all udev info for a device
udevadm info --query=all --name=/dev/IBMtape0
Get device properties
udevadm info --query=property --name=/dev/IBMtape0
Get device serial number
udevadm info --query=property --name=/dev/IBMtape0 | grep -E 'ID_SCSI_SERIAL|ID_SERIAL_SHORT'
Walk sysfs attributes
udevadm info --attribute-walk --name=/dev/IBMtape0 | head -60
Check /proc for tape info
cat /proc/scsi/IBMtape 2>/dev/null

FC HBA Ports

List all FC HBA ports (WWPN + state)
for host_dir in /sys/class/fc_host/host*; do
    host=$(basename "$host_dir")
    wwpn=$(cat "${host_dir}/port_name" 2>/dev/null)
    state=$(cat "${host_dir}/port_state" 2>/dev/null)
    echo "  ${host}: WWPN=${wwpn}  State=${state}"
done

Kernel Messages

Recent dmesg — tape-related entries
dmesg | grep -i -E "tape|IBMtape|lin_tape|changer" | tail -20
🔗 Persistent Device Binding (udev)

Create stable symlinks under /dev/tsmtape/ so Storage Protect always finds the same device paths, even if kernel device numbering changes.

udev Rules Template

File: /etc/udev/rules.d/99-ibm-tape-persist.rules
# IBM Storage Protect — Persistent tape device symlinks
# Library: Spectra Logic with IBM LTO drives (dual-pathed)

# --- Medium Changer (Library) ---
KERNEL=="IBMchanger[0-9]*", ATTRS{serial}=="LIBRARY_SERIAL", SYMLINK+="tsmtape/library0"

# --- Drive 01 (serial: DRIVE01_SERIAL) ---
KERNEL=="IBMtape[0-9]*", ATTRS{serial}=="DRIVE01_SERIAL", ENV{ID_PATH}=="*-lun-0", SYMLINK+="tsmtape/drive01_path0"
KERNEL=="IBMtape[0-9]*", ATTRS{serial}=="DRIVE01_SERIAL", ENV{ID_PATH}=="*-lun-1", SYMLINK+="tsmtape/drive01_path1"

# --- Drive 02 (serial: DRIVE02_SERIAL) ---
KERNEL=="IBMtape[0-9]*", ATTRS{serial}=="DRIVE02_SERIAL", ENV{ID_PATH}=="*-lun-0", SYMLINK+="tsmtape/drive02_path0"
KERNEL=="IBMtape[0-9]*", ATTRS{serial}=="DRIVE02_SERIAL", ENV{ID_PATH}=="*-lun-1", SYMLINK+="tsmtape/drive02_path1"

Apply Rules

Create symlink directory
mkdir -p /dev/tsmtape
Reload udev rules and trigger
udevadm control --reload-rules && udevadm trigger
Verify symlinks created
ls -la /dev/tsmtape/
Check symlink targets
readlink -f /dev/tsmtape/*

Useful udev Match Attributes

AttributeDescription
ATTRS{serial}Drive serial number (most reliable key)
ATTRS{vendor}e.g. 'IBM'
ATTRS{model}e.g. 'ULT3580-HH9' (LTO-9)
ENV{ID_PATH}HBA path identifier — use for dual-path matching
ENV{ID_SCSI_SERIAL}SCSI serial string
ENV{ID_SERIAL_SHORT}Short serial number
⚙️ Storage Protect Configuration

dsmadmc commands to define the library, drives, device class, and storage pool in IBM Storage Protect.

⚠ Placeholder Values Replace SERVERNAME, ADMIN, PASSWORD, SPECTRALIB, element addresses, and serial numbers with your actual values before running these commands.

Connect to Storage Protect

Locate dsmadmc binary
command -v dsmadmc || ls /opt/tivoli/tsm/client/ba/bin/dsmadmc /opt/tivoli/tsm/server/bin/dsmadmc 2>/dev/null
Connect to SP admin console
dsmadmc -id=ADMIN -pa=PASSWORD -displ=list

Define Library

Define SCSI library
DEFINE LIBRARY SPECTRALIB LIBTYPE=SCSI
Define library path
DEFINE PATH SERVERNAME SPECTRALIB SRCTYPE=SERVER DESTTYPE=LIBRARY DEVICE=/dev/tsmtape/library0 ONLINE=YES

Define Drives

Define drive (ELEMENT= is the library element address)
DEFINE DRIVE SPECTRALIB DRIVE01 ELEMENT=256 CLEANFREQUENCY=ASNEEDED
Define primary drive path
DEFINE PATH SERVERNAME DRIVE01 SRCTYPE=SERVER DESTTYPE=DRIVE LIBRARY=SPECTRALIB DEVICE=/dev/tsmtape/drive01_path0 ONLINE=YES
Define secondary drive path (dual-path)
DEFINE PATH SERVERNAME DRIVE01 SRCTYPE=SERVER DESTTYPE=DRIVE LIBRARY=SPECTRALIB DEVICE=/dev/tsmtape/drive01_path1 ONLINE=YES

Audit & Configure

Audit library (reads barcodes)
AUDIT LIBRARY SPECTRALIB CHECKLABEL=BARCODE
Define LTO device class
DEFINE DEVCLASS LTO_DC DEVTYPE=LTO LIBRARY=SPECTRALIB FORMAT=DRIVE MOUNTRETENTION=60 MOUNTWAIT=60 MOUNTLIMIT=DRIVES
Define storage pool
DEFINE STGPOOL LTO_POOL LTO_DC MAXSCRATCH=500 COLLOCATE=FILESPACE REUSEDELAY=1
Label and check in tapes from barcodes
LABEL LIBVOLUME SPECTRALIB SEARCH=YES LABELSOURCE=BARCODE CHECKIN=SCRATCH OVERWRITE=NO
🔀 Dual-Path Failover

Configure and test control/data path failover with lin_tape's alternate_pathing feature.

⛔ Warning — Maintenance Window Required Failover testing temporarily takes drive paths offline. Run during a maintenance window only!

Check Alternate Pathing

Check alternate_pathing module parameter
cat /sys/module/lin_tape/parameters/alternate_pathing 2>/dev/null
Verify all paths are online
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY PATH FORMAT=DETAILED" | grep -E "Source Name|Destination Name|Device|Online"

Failover Test

Take path0 offline (simulate failure)
dsmadmc -id=ADMIN -pa=PASSWORD "UPDATE PATH SERVERNAME DRIVE01 SRCTYPE=SERVER DESTTYPE=DRIVE LIBRARY=SPECTRALIB DEVICE=/dev/tsmtape/drive01_path0 ONLINE=NO"
Query drive status (should show active on path1)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY DRIVE SPECTRALIB DRIVE01 FORMAT=DETAILED"
Restore path0 online
dsmadmc -id=ADMIN -pa=PASSWORD "UPDATE PATH SERVERNAME DRIVE01 SRCTYPE=SERVER DESTTYPE=DRIVE LIBRARY=SPECTRALIB DEVICE=/dev/tsmtape/drive01_path0 ONLINE=YES"
✅ Verification & Testing

Query commands to verify the full configuration is working correctly in Storage Protect.

Query Library & Drives

Query library (detailed)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY LIBRARY SPECTRALIB FORMAT=DETAILED"
Query library path
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY PATH SERVERNAME SPECTRALIB SRCTYPE=SERVER DESTTYPE=LIBRARY"
Query all drives
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY DRIVE SPECTRALIB"
Query all paths (detailed)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY PATH FORMAT=DETAILED"

Query Device Class & Pool

Query device class (detailed)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY DEVCLASS LTO_DC FORMAT=DETAILED"
Query storage pool (detailed)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY STGPOOL LTO_POOL FORMAT=DETAILED"
Query library volumes (tape inventory)
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY LIBVOLUME SPECTRALIB"
Query volumes in pool
dsmadmc -id=ADMIN -pa=PASSWORD "QUERY VOLUME * STGPOOL=LTO_POOL"

Test Archive

Test archive a file to tape
dsmc archive /tmp/testfile -se=SERVERNAME
🏥 Diagnostics & Troubleshooting

Quick diagnostic commands and a full status script for when things go wrong.

Full Status Script

Run full driver & device diagnostic check
echo "=== lin_tape Status ==="
lsmod | grep lin_tape
modinfo lin_tape 2>/dev/null | head -10
rpm -qa | grep -i lin_tape
echo ""
echo "=== Devices ==="
ls -la /dev/IBMtape* /dev/IBMchanger* 2>/dev/null
echo ""
echo "=== Symlinks ==="
ls -la /dev/tsmtape/ 2>/dev/null
echo ""
echo "=== FC HBAs ==="
for h in /sys/class/fc_host/host*; do
    [ -d "$h" ] || continue
    echo "$(basename $h): WWPN=$(cat $h/port_name 2>/dev/null) State=$(cat $h/port_state 2>/dev/null)"
done
echo ""
echo "=== dmesg (tape) ==="
dmesg | grep -i -E "tape|IBMtape|lin_tape|changer" | tail -20

Error Checks

Check for SCSI reservation conflicts
dmesg | grep -i "reservation conflict"
Check SCSI errors and timeouts
dmesg | grep -i -E "scsi|error|timeout" | tail -20
Verify kernel module parameters
cat /sys/module/lin_tape/parameters/* 2>/dev/null
Check all lin_tape systemd service statuses
systemctl status lin_tape lin_taped lin_tape_load 2>/dev/null
